### Daily Notes

This folder contains a Markdown file for each day. Each entry follows the format:

```markdown
# [YYYY-MM-DD] - Daily Reflections

## Topics Covered
- Topic 1
- Topic 2

## Key Insights
- Insight 1
- Insight 2

## Challenges and Plans
- Challenge 1
- Strategy for tomorrow
```

To start your daily log, use a template file from the `templates` directory.
